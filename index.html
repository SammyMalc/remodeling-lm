Connexion au Google Sheet
Une seule configuration suffit. Suivez ces étapes :
1
Allez sur sheets.google.com → créez un nouveau fichier → nommez-le Remodeling LG.
2
Cliquez Partager → Tout le monde avec le lien → rôle Éditeur.
3
Copiez l'ID depuis l'URL : docs.google.com/spreadsheets/d/[ID ICI]/edit
ID du Google Sheet
Partagez ce fichier HTML à vos collègues — ils saisiront le même ID une seule fois.
